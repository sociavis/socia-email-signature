// Installation

Apple Mail

  1. Open Mail → Settings → Signatures
  2. Click + to create a new signature
  3. Quit Mail completely (Cmd+Q)
  4. Go to Finder → Go to Folder: ~/Library/Mail/V10/MailData/Signatures/
  5. Open the newest .mailsig in TextEdit
  6. Delete everything, paste contents of apple-mail.html
  7. Save and reopen Mail
↓ Download apple-mail.html

Gmail / Outlook

  1. Gmail: Settings → See all settings → Signature → switch to HTML mode
  2. Paste the full HTML source of apple-mail.html
  3. Outlook: Insert → Signature → paste HTML
  4. Animated GIF loads from Vercel CDN — no inline assets needed
  5. Tip: ensure "display external images" is enabled in client settings
